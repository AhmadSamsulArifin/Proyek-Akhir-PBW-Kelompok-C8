# 📌 Deskripsi Website

<details>
<summary>🌟 Gambaran Umum</summary>

<br>

**Mahakam Lampion Garden Samarinda** adalah website promosi dan informasi wisata berbasis web yang dirancang untuk membantu penyampaian informasi mengenai tempat wisata Mahakam Lampion Garden secara lebih lengkap, terpusat, dan mudah diakses oleh masyarakat.

Website ini dibuat karena Mahakam Lampion Garden belum memiliki website resmi sebagai pusat informasi wisata. Informasi mengenai lokasi, fasilitas, harga tiket, kegiatan wisata, dan reservasi sebelumnya masih bergantung pada media sosial seperti Instagram dan Facebook, sehingga informasi yang diterima calon pengunjung belum selalu lengkap dan terstruktur.

Melalui website ini, pengunjung dapat melihat informasi wisata, galeri foto, ulasan pengunjung, lokasi, serta menghubungi pengelola melalui WhatsApp untuk reservasi acara atau informasi lebih lanjut. Selain itu, website ini juga menyediakan halaman admin yang dapat digunakan oleh pengelola untuk memperbarui data wisata, galeri, ulasan, informasi tiket, jam operasional, dan kontak.

Website ini terdiri dari dua bagian utama, yaitu:

- **Website Publik**, yang dapat diakses oleh masyarakat umum sebagai media promosi dan informasi wisata.
- **Website Admin**, yang digunakan oleh pengelola untuk mengelola seluruh konten website.

Dengan adanya website ini, Mahakam Lampion Garden diharapkan memiliki media informasi resmi yang lebih rapi, modern, dan mendukung promosi wisata Kota Samarinda.

</details>

<details>
<summary>🎯 Tujuan Pembuatan Website</summary>

<br>

Tujuan utama dari pembuatan website ini adalah:

- merancang dan mengimplementasikan website resmi Mahakam Lampion Garden sebagai media promosi dan pusat informasi wisata;
- menyediakan informasi lengkap mengenai profil wisata, harga tiket, jam operasional, fasilitas, galeri, lokasi, dan informasi pendukung lainnya;
- membantu calon pengunjung memperoleh informasi wisata secara jelas, terpusat, dan mudah diakses;
- mendukung promosi Mahakam Lampion Garden agar lebih dikenal oleh masyarakat luas;
- menyediakan fitur ulasan pengunjung sebagai sarana masukan dan penilaian terhadap tempat wisata;
- menyediakan akses komunikasi melalui WhatsApp untuk kebutuhan reservasi acara atau informasi lebih lanjut;
- menyediakan dashboard admin untuk mengelola informasi website seperti data wisata, galeri, ulasan, dan pengaturan informasi.

</details>

# 🔄 Alur Sistem Website

<details>
<summary>📌 Klik untuk melihat alur sistem website</summary>

<br>

Berikut gambaran alur kerja website secara umum:

1. Pengunjung membuka website Mahakam Lampion Garden.
2. Sistem menampilkan halaman beranda.
3. Pengunjung dapat memilih menu Beranda, Wisata, Galeri, Ulasan, Tentang, atau Kontak.
4. Pengunjung dapat melihat informasi wisata, fasilitas, harga tiket, jam operasional, dan galeri.
5. Pengunjung dapat melihat dan mengisi ulasan.
6. Pengunjung dapat menekan tombol WhatsApp untuk reservasi atau informasi lebih lanjut.
7. Pengunjung dapat menekan tombol Google Maps untuk melihat lokasi wisata.
8. Admin membuka halaman login admin.
9. Admin login menggunakan akun yang valid.
10. Sistem menampilkan dashboard admin.
11. Admin dapat mengelola data wisata, galeri, ulasan, dan pengaturan informasi.
12. Informasi yang diperbarui admin akan ditampilkan pada website publik.

</details>

# 📲 Cara Menggunakan Website

<details>
<summary>👤 Jika sebagai Pengunjung</summary>

<br>

1. Buka website Mahakam Lampion Garden.
2. Lihat informasi pada halaman beranda.
3. Pilih menu **Wisata** untuk melihat spot wisata.
4. Klik detail wisata untuk melihat informasi lebih lengkap.
5. Buka menu **Galeri** untuk melihat foto wisata.
6. Buka menu **Ulasan** untuk melihat dan memberikan ulasan.
7. Buka menu **Tentang** untuk membaca profil Mahakam Lampion Garden.
8. Gunakan tombol **WhatsApp** untuk menghubungi pengelola.
9. Gunakan tombol **Google Maps** untuk melihat lokasi wisata.

</details>

<details>
<summary>🛠️ Jika sebagai Admin</summary>

<br>

1. Buka halaman login admin.
2. Masukkan akun admin.
3. Masuk ke dashboard admin.
4. Kelola data wisata melalui menu **Kelola Wisata**.
5. Kelola foto melalui menu **Kelola Galeri**.
6. Moderasi ulasan melalui menu **Kelola Ulasan**.
7. Perbarui harga tiket, jam operasional, alamat, dan kontak melalui menu **Pengaturan Informasi**.
8. Logout setelah selesai menggunakan dashboard admin.

</details>
